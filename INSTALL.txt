If you want to get up and running as quick as possible, you will probably
be better off using the webstart version instead of this manual installation.

To run the webstart version, just point your browser to:

http://bookscanwizard.sourceforge.net/run



If you really do want to do this manually, follow the directions below.


--Prerequisites--

You will need to install the following before running the Book Scan Wizard:

Java 1.6
http://www.java.com/en/download/

Java Advanced Imaging  (JAI) 1.1.3
https://jai.dev.java.net/binary-builds.html

JAI Tools 1.1:
https://jai-imageio.dev.java.net/binary-builds.html


-- Running --

To run the application, run either:
java -Xmx1024 -jar BookScanWizard.jar

Or you can use run of the included scriptss. In the bin directory there are scripts bsw.cmd, bsw, and bsw.sh.
If you are running windows, you can just run "bsw".  If you are running linux or are on a mac, you can run bsw.sh or bsw.

command arguments:


run 

bsw -?

To get the list of command arguments.
